Input-Stars Component

A simple Angular Reactive Form element for capturing ratings. The element uses the same conventions as other form elements like "input" to build up a rich form experience.

Usage

Add a rating element to your form:

Component

this.myForm = this.fb.group({ myStars: [0] })

Template

<form name="myForm" id="myForm" [formGroup]="myForm">
  <p>Rate It!</p>
  <input-stars formControlName="myStars"
    [max]="5"
    ></input-stars>
  <p>You've given it a {{ myForm.get('myStars').value }}!<br>
  The rating is {{ myForm.get('myStars').pristine ? 'untouched' : 'touched' }}</p>
</form>
