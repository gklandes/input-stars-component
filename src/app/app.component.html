Input-Stars Component Demo

InputStarsComponent integrates easily with both Reactive and Template-driven Angular forms. This demo shows how to add an "input-stars" element to a Reactive Form.

Component

this.myForm = this.formBuilder.group({ myStars: [null, [ Validators.required ]] })

Template

<form name="myForm" id="myForm" [formGroup]="myForm">
  <p>Rate It!</p>
  <input-stars formControlName="myStars"
    [max]="5"
    ></input-stars>
  <p>You've given it {{ myForm.get('myStars').value || 'no' }}!</p>
</form>
